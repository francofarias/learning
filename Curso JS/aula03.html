Tratamento de Dados

O método prompt() como vimos mostra na tela uma caixa de alerta com um campo para preenchimento e retorna o valor digitado pelo usuário

Podemos assim armazenar esse valor em uma variável como no exemplo abaixo.

let nome = window.prompt("Qual é seu nome?")

O código acima gera a seguinte resposta do navegador:

Nesse exemplo preenchemos o campo com "Franco", logo esse valor será atribuido a variável nome

Para exercitar a manipulação de dados com variáveis, vamos usar o método alert() que vimos anteriormente, mas apresentando o valor da variável nome na mensagem do alert()

Para isso usamos o seguinte código:

window.alert("olá " + nome)

Dentro das aspas está uma mensagem normal como a que digitamos anteriormente, e fora das aspas está nossa variável nome, e para que possamos juntar o texto que digitamos com a variável usamos o sinal de +, o resultado fica assim:
