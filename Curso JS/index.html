Aula 01

Vamos começar aprendendo três códigos simples para nos familiarizarmos com o javascript.

alert

O código: windonw.alert('') exibe na tela uma "janela" acima do site com a informação contida dentro das aspas.

Teste:

windonw.alert('Minha primeira mensagem')

confirm

O código: windonw.confirm('') exibe na tela uma "janela" acima do site com a informação contida dentro das aspas e solicita a corfirmação (ok) ou não (cancelar) da mensagem.

Teste:

windonw.confirm('Está gostando do curso?')

Você deve ver um resultado como:

prompt

O código: windonw.prompt('') exibe na tela uma "janela" acima do site com a informação contida dentro das aspas e solicita o preenchimento do campo de texto conforme instrução da mensagem.

Teste:

windonw.prompt('Qual é o seu nome?')
