Anotações Curso JS: Aula 02

Variáveis

Para criar variáveis no JS podemos utilizar o var ou let, eles tem diferenças que vamos ver mais a frente, mas por ora daremos prferência ao let.

As variáveis possuem identificadores, nomes, os identificadores precisam respeitar as seguintes regras:

Só podem começar com:letras, $ ou _;
Não podem começar com números;
É possível utilizar letras e números;
É possível utilizar acentos e símbolos;
Não podem conter espaços;
Não podem ser palavras reservadas.

Exemplo de criação de variáveis:

let nome
let idade
let sexo

Dicas

JS diferência maiúscula de minúscula, por tanto "a" é dirente de "A";

Use nomes coerentes para suas variáveis, que informem o proposito e até o contexto daquela variável, ex: nome, nomeTurma, cidade, idade, dataNascimento;

Tipos

As variáveis guardam valores/dados, esses valores possuem tipos, como numero ou texto (string) por exemplo, segue uma relação dos tipos de dados em JS:

number;
infinity;
NaN;
string;
boolean;
null;
undefined;
objet;
Array;
function.

Para verificar o tipo de um dado no JS você pode utilizar o comando typeof, exemplo:

> typeof 5
< 'number'

O tipo de 5 é um número.
